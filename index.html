Array, Matrices o Arreglos

Es un conjunto de datos

Un array o matriz es una estructura de datos que nos permite almacenar varios valores en una sola variable.

Los array te permiten almacenar diferentes tipos de datos en un solo lugar en vez de tener diferentes variables que esten dispersas

Para acceder a los datos dentro de un array, siempre se empieza desde 0 haciendo referencia que es el dato en primera posición.
Como en este ejemplo, el dato en la posición 0 es "zapatillas" y el dato en la posición 1 es "camiseta" y así sucesivamente.
Con un alert podemos mostrar el dato en la posición [1] que nos dara como resultado "camiseta". Tambien podemos mostrar el dato en la consola con console.log.

Propiedad length: Nos permite saber cuantos elementos tiene un array

Metodo push: Nos permite agregar un nuevo elemento dentro de un array con el que vamos a trabajar

Metodo unshift: Nos permite agregar un nuevo elemento al comienzo del array

Metodo pop: Nos permite eliminar el ultimo elemento del array

Metodo shift: nos permite eliminar el primer elemento del array
